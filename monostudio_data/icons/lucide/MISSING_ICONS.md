# Lucide icons – đã bổ sung

Các icon từng thiếu đã được tải (Iconify API, Lucide) và thêm vào thư mục:

| Icon           | File               | Dùng ở |
|----------------|--------------------|--------|
| arrow-right    | `arrow-right.svg`  | main_view.py |
| chevron-left   | `chevron-left.svg` | calendar_date_picker.py |
| folder-plus    | `folder-plus.svg`  | main_window.py |
| package-open   | `package-open.svg` | main_view.py |

Tất cả đều dùng `stroke="currentColor"` để app tô màu theo theme.
